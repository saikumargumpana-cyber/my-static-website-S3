# 🌐 Static Website Hosting using Amazon S3

This project demonstrates how to host a static website using an AWS S3 bucket.

---

## 📌 Project Overview

This is a simple static website built using:

- HTML
- CSS
- JavaScript

The website is deployed using Amazon S3 Static Website Hosting feature and made publicly accessible.

---

## 🛠️ Technologies Used

- Git
- GitHub
- HTML5
- CSS3
- JavaScript
- AWS S3

---

static-website-s3/
│

---

## 🚀 Deployment Steps

1. Created an S3 bucket
2. Disabled "Block all public access"
3. Uploaded website files
4. Enabled Static Website Hosting
5. Configured bucket policy for public read access
6. Accessed website using S3 website endpoint URL

---

## 🔗 Live Website

S3 Website Endpoint:

http://your-bucket-name.s3-website-region.amazonaws.com

(Replace with your actual S3 website URL)
